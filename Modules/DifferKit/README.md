# DifferKit

Shared utilities, extensions, and helpers for the Differ application.

## Overview

This module provides common utilities used across services and UI. It depends only on `DifferCore`.

## Contents

### Planned Utilities

- File system helpers
- Common extensions (String, URL, NSImage)
- Logging utilities
- Date formatting
- Color utilities
